kadelnode

NAME
	kadelnode - unregisters nodes

SYNOPSIS
	kadelnode -m|--machine hostname

DESCRIPTION
	kadelnode unregisters nodes and their disk and partition features from ka deployment system.

OPTIONS
	-m, --machine hostname
		specifies a host. This option can be used more than once to specify several hosts.

RETURN VALUE
       kadelnode returns 0 if it fails. Otherwise, it returns 1.

EXAMPLE
	kadelnode -m node	# remove all information relative to node from ka deployment system
