kacreateenv

NAME
	kacreateenv - creates and registers a new environment

SYNOPSIS
	kacreateenv -e|--environment environment_name -v|--version version [-X|--exclude-from file] [-z|--gzip] [--output-directory output-directory] [--root-directory root_directory] [--author email_address] [--description "description"] [--file-base file] [--file-site file] [--size size] [--kernel-path kernel_path] [--fdisk-type fdisk_type_number] [--file-system file_system_type]

DESCRIPTION
	kacreateenv creates and registers a new environment from a running system one.

OPTIONS
	-e, --environment environment_name 
		name of the environment image output file.

	-v, --version version
		version of the environment ; 1 by default.

	-X, --exclude-from file
		excludes files named into file from the generated environment.

	-z, --gzip
		filter the archive through gzip.

	--output-directory output_directory
		sets the output directory for the generated environment.

	--root-directory root_directory
		sets the root directory for the generated environment.
	
	--author environment_author
		email address of the image's author.

	--description "environment_description"
		brief description of the environment to create.

	--file-base base_file_locator
		absolute path to the valid basic system image.

	--file-site site_file_locator
		absolute path to the validated image specialization file.

	--size decompacted_environment_size
		size of the deployed environment in Mbytes. By default, it is the sum of the base and specialization file size.

	--kernel-path kernel_absolute_path
		absolute path of the kernel in the submitted environment.

	--fdisk-type fdisk_type_number
		fdisk type of the target partition. By default, type 83 (Linux) is used.

	--file-system file_system_type
		file system associated to the environment. By default, ext2 file system is used.

RETURN VALUE
       kacreateenv returns 0 if it fails. Otherwise, it returns 1.

EXAMPLE
	cd /tmp
	kacreateenv -e environment_name -X /path/to/file_to_exclude.txt --author toto --description "toto's special image" --size 650
