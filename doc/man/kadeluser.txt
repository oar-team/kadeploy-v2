kadeluser

NAME
	kadeluser - revokes user granted rights on given nodes and partitions (root command)

SYNOPSIS
	kadeluser [-l|--login login] [-m|--machine hostname] [-p|--partition partition]

DESCRIPTION
	kadeluser revokes deployment rights to the specified user on the given partition

OPTIONS
	-l, --login login
		the login of the user. This option can be used more than once to specify several users. 
	-m, --machine hostname
		specifies a host. This option can be used more than once to specify several hosts. By default, the deployment rights are revoked for all the nodes.
	-p, --partition partition
		the target partition. This option can be used more than once to specify several partitions. By default, the deployment rights are revoked for all the partitions.

RETURN VALUE
       kadeluser returns 0 if it fails. Otherwise, it returns 1.

EXAMPLE
	kadeluser -l titi -p hda3			# revokes rights to user titi on all nodes on partition hda3
	kadeluser -m node1 -p hdb6			# revokes rights to all the users on node1 partition hdb6
	kadeluser -l toto -m node1 -m node2 -p hda1     # revokes rights to user toto on node1 and node2 partition hda1
	etc.
