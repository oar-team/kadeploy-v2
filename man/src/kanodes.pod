=head1 NAME

kanodes - registers and remove nodes, disk, partition.

=head1 SYNOPSIS

B<kanodes>  B<[-add]>  [B<-del> hostname]  [B<-listnode>]  [B<-listpartition>]  [B<-h>]

=head1 DESCRIPTION

kanodes registers nodes and their disk and partition features for them to be available to the deployment system on a given cluster. It is used at the very begining to describe a whole cluster (after having completed the installation for instance) or later when new nodes are added to a cluster already managed by the deployment system. 

It use two files named B</etc/kadeploy/clusternodes.conf>  B</etc/kadeploy/clusterpartition.conf>.

=head1 OPTIONS

=over

=item B<-add>

Loads the configuration files.

=item B<-del>

Deletes a specified node or a list of nodes.

=item B<-listnode>

Lists registered nodes.

=item B<-listpartition>

Lists registered partitions.

=item B<-h>

Prints help screen.

=back
